准备工作：

Hexo

Hexo 基于 node.js，所以首先我们要在本地安装node.js和git，除了安装位置其他全部默认就行。

下载之后找到cmd，分别输入 node -v、npm -v 和 git --version，如果出现版本号说明安装成功。

Github 的账号注册

需要准备好一个邮箱和一个科学上网工具，其他不用多说。
